# UniversalDialog
通用的对话框，可以弹出1个按钮 2个按钮 最多3个按钮
================================================
# 如何使用它？
先在 build.gradle 的 repositories 添加:
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

然后在dependencies添加:
```
	dependencies {
	        compile 'com.github.everyline:UniversalDialog:v1.0'
	}
```
